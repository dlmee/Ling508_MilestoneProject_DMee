Methods

Generate database

NOTE: This will take a few minutes, it's doing a lot of processing under the hood.

You will have to run this first, before you generate a post request. Otherwise the database will be largely empty.

Currently the scrape is pointed at 'http://host.docker.internal:8000/Scrapes/GM_TPaC.html' this means you will need to have a local server running at this directory

Currently this is set up as a GET request type, but this could easily be retooled as a POST to target different databases.

Endpoint: http://localhost:5000/generatedb

Request type: GET

Response: 

  {
    "msg": "success"
  }

Find the senses of a word.

Endpoint: http://localhost:5000/find_sense

Request type: POST

Request body: JSON with field "word", e.g.

Try this out here: HTML Form 

  {
    "word": "beauty"
  }

Response: JSON list of dictionaries with word form, sense, vector, and example sentences, e.g. 

{
        "examples": [
            " Fearing at last that he was rude  he turned them away; and  behold  he was in a room that was for beauty marvellous"
        ],
        "sense": 2,
        "surface": "beauty",
        "vector": [
            [
                "fearing",
                1
            ],
            [
                "behold",
                1
            ],
            [
                "marvellous",
                1
            ]
        ]
    }
